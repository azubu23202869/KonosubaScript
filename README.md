# KonosubaScript


套件安裝
```
    pip install opencv-python==3.* -i https://pypi.tuna.tsinghua.edu.cn/simple/
    pip install matplotlib -i https://pypi.tuna.tsinghua.edu.cn/simple/
    pip install uiautomator2 
    pip install numpy
```

初始化u2
```
    python -m uiautomator2 init #CMD運行
```

運行
``` 
    python main.py
```
